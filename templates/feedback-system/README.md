# 意见反馈系统界面模板

这是一个独立静态模板，用于讨论意见反馈系统的产品界面和业务流程，不影响现有 Next.js 页面。

## 文件

- `index.html`：用户端反馈页和管理员处理页的静态结构。
- `styles.css`：模板样式。
- `script.js`：视图切换和列表选中态。

## 使用方式

直接用浏览器打开 `index.html` 即可查看，不需要启动开发服务器。

## 覆盖流程

- 用户提交反馈。
- 用户查看反馈记录。
- 用户继续补充信息。
- 管理员查看反馈队列。
- 管理员筛选和搜索反馈。
- 管理员回复用户。
- 管理员修改反馈状态。
- 管理员查看关联任务、用户、订单和内部备注。

## 后续接入建议

正式接入项目时，可以拆成两个页面：

- 用户端：`/tickets`
- 管理端：`/admin/tickets`

核心数据结构建议沿用 `feedback_ticket` 和 `feedback_message` 两张表。
